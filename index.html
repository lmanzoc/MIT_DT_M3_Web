Tu peudes editar esta demo desde Github

Como usar el editor:

Pega tu código en el archivo index.html empleando el editor de Github.
Edita cualquier texto del archivo en tiempo real. 

Dar click en el botón Commit para guardar tu código html en la rama Master.

Algunas características muy usadas:

  1. Editor de código fuente interactivo
  2. HTML legible
  3. Conversión de Word a HTML
  4. Buscar y Reemplazar
  5. Generador Lorem-Ipsumr
  6. Conversión de Tabla a DIV
